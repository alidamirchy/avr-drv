Compute value for UBRR.

Parameters:
uiBaudRateDesired baud rate.
uiClkClock frequency in Hz.
ubTolMismatch tolerance. 2 should be use, witch mean 2%.
modeWhether it is Asyn or Sync USART mode.
puwUbrrPointer to where UBRR value will be saved.
Returns:
On success, value can be 0 or 1. If value is 0, X2 mode shall be desactivated and if it is 1, it shall be activated. Value -1 indicate that an error occurred. See errno for detail on error.
