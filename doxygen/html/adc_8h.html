void adc_left_adjust ( _Bool  adjust)

Select how conversion result will be stored in register.

For 8-bits result, use:

  • adc_left_adjust(true)
  • adc_read_high()

For 10-bits result, use:

  • adc_left_adjust(false)
  • adc_read()
Parameters:
adjustAlignment of the result register.
Note:
Devices with 8-bits ADC do not support this function.
